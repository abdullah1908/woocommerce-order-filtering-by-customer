== Description ==

This plugin creates a new tab in My Account Page named as Order History & allows user to filter the own orders based on month.

== Installation ==

1. Upload `woo-order-filtering-by-customer` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
